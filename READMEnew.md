## Vita's Ecommerce Project

#### Colors:

- Red -> #E2352D
- Green -> #234125
- Dark Gray -> #333333
- Light Gray -> #EEEEEE

#### Scripts to run at backend folder:

- python3 -m venv venv
- . ./venv/bin/activate
- pip install -r requirements.txt
- python manage.py runserver


#### Scripts to run at frontend folder:

- yarn install
- yarn dev
